# x264

This crate gives you safe but still extremely fast H.264 encoding.

## Usage

1. Add `x264 = "0.2"` to your `Cargo.toml`.
2. Look at an [example](examples) and copy it.
3. Read some [documentation](https://docs.rs/x264).
